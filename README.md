# AI Course Recommendation System

## Overview

This project is a Content-Based Recommendation System that recommends Coursera courses based on user interests using Natural Language Processing (NLP).

The recommendation engine uses:

- TF-IDF Vectorization
- Cosine Similarity
- Python
- Scikit-learn

## Features

- User enters interests
- Converts text into TF-IDF vectors
- Calculates cosine similarity
- Displays Top 5 recommended courses

## Dataset

Coursera Courses Dataset from Kaggle

Columns used:

- Title
- Organization
- Skills
- Ratings
- Review counts
- Metadata

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
